# Frag Platform

Платформа для коллекционеров парфюма: каталог личной коллекции + реальная рыночная
стоимость как актива + соц-граф + цены/алерты.

Рабочая метафора: **«Vivino × Delectable для парфюма»** с портфельным трекером
в стиле кроссовок/часов.
